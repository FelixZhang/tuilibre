# Tuilibre v0.1.0 发布说明

## 🎉 首次发布

我们很高兴地宣布 **Tuilibre v0.1.0** 的正式发布！这是一个用于管理 calibre 数字图书馆的现代化 TUI（终端用户界面）工具。

## ✨ 核心特性

### 🔍 智能搜索系统
- **实时搜索**：输入即搜，提供类似 fzf 的流畅搜索体验
- **多字段搜索**：支持书名、作者、标签、路径搜索
- **拼音搜索**：支持通过拼音搜索中文书籍（如输入 `lishi` 搜索历史类书籍）
- **路径搜索**：按文件夹组织和搜索书籍，适合分类管理

### 📚 图书馆管理
- **自动发现**：智能搜索系统中的 calibre 图书馆
- **历史记录**：自动记录使用过的图书馆，按使用频率排序显示
- **无缝切换**：使用 ESC 键在不同图书馆间快速切换
- **跨平台支持**：完全支持 Linux、macOS、Windows 系统

### 🎯 用户友好界面
- **完全中文界面**：本地化的中文 TUI 界面，符合中国用户习惯
- **Vim 风格操作**：支持 j/k 移动，/ 搜索，ESC 返回等快捷键
- **实时反馈**：所有操作都有即时的视觉反馈
- **丰富信息显示**：展示书籍标题、作者、路径、标签等完整信息

### ⚡ 卓越性能
- **异步操作**：基于 tokio 的非阻塞数据库操作
- **快速启动**：优化的数据库连接和书籍加载
- **内存高效**：流式加载，支持大型图书馆（数千本书）
- **响应迅速**：实时搜索和界面更新，无延迟感

## 🚀 新手指南

### 快速安装

```bash
# 1. 克隆仓库
git clone <repository-url>
cd tuilibre

# 2. 构建应用
cargo build --release

# 3. 启动应用
./target/release/tuilibre
```

### 基本使用

```bash
# 方式1：启动库选择器（推荐）
tuilibre

# 方式2：直接打开指定图书馆
tuilibre /path/to/your/calibre/library

# 方式3：使用标志语法
tuilibre --library /path/to/your/calibre/library
```

### 搜索示例

```bash
# 中文拼音搜索
lishi        → 搜索历史类书籍
wenxue       → 搜索文学类书籍
keji         → 搜索科技类书籍

# 路径搜索
programming  → 搜索编程相关书籍
fiction      → 搜索小说类书籍
tech         → 搜索技术类书籍
2023         → 搜索2023年书籍

# 混合搜索
任何关键词都会在书名、作者、标签、路径中搜索
```

## 📖 完整功能指南

### 键盘快捷键

#### 书籍浏览模式
- `↑↓` 或 `j/k`：上下导航书籍列表
- `Enter`：查看选中书籍的详细信息
- `/`：进入实时搜索模式
- `ESC`：返回库选择界面（🆕新功能）
- `q`：退出应用

#### 搜索模式
- 直接输入搜索关键词进行实时搜索
- `j/k` 或 `↑↓`：在搜索结果中导航
- `Enter`：查看搜索结果中的书籍详情
- `ESC`：退出搜索，返回完整书籍列表
- `q`：退出应用

#### 库选择界面
- `↑↓` 或 `j/k`：导航可用的图书馆列表
- `Enter`：选择并打开选中的图书馆
- `q`：退出应用

#### 书籍详情视图
- `ESC`：返回书籍列表或搜索结果
- `Enter`：使用系统默认应用打开书籍文件
- `q`：退出应用

### 高级功能

#### 🏠 历史记录系统
- 自动记录使用的图书馆
- 按使用频率和最近使用时间排序
- 显示图书馆名称、路径、书籍数量和使用次数
- 星号 ⭐ 标识历史中的图书馆

#### 📁 路径搜索支持
- 支持按文件夹结构搜索书籍
- 适合按类型、年份、语言等分类管理
- 在书籍列表中显示路径信息
- 支持部分路径匹配搜索

#### 🔤 文件集成
- 使用 xdg-open（Linux）、open（macOS）、start（Windows）打开书籍
- 自动解析 calibre 文件结构
- 支持所有 calibre 支持的书籍格式

## 🔧 系统要求

### 依赖项
- **Rust 1.70+**：现代化 Rust 编程语言
- **calibre**：数字图书馆管理系统

- **Linux**：所有主流发行版

- **Windows**：Windows 10+ 版本

## 🛠️ 故障排除

### 常见问题

**Q: 应用启动时找不到数据库**
A: 确保指定目录包含 calibre 的 `metadata.db` 文件

**Q: 搜索结果为空**
A: 尝试使用不同的搜索关键词，检查书籍标题、作者、路径、标签

**Q: 路径显示乱码**
A: 确保终端支持 UTF-8 编码

**Q: 无法打开书籍文件**
A: 检查书籍文件是否存在于计算上，确认系统有默认的文件打开应用
